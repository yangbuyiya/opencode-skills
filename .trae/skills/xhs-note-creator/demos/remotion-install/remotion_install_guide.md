# 🎬 什么是 Remotion？

在 React 中创建视频的强大工具，可以用代码生成视频，超适合做动画、宣传片、教程视频！

开发者福音来了，再也不用用笨重的视频剪辑软件了～

---

# 📦 安装步骤

### 1️⃣ 执行安装命令

```bash
npx skills add remotion-dev/skills
```

系统会提示需要安装 skills@1.4.4，输入 `y` 确认安装。

---

### 2️⃣ 选择 AI 助手

支持 Amp、Cline、Cursor、GitHub Copilot、OpenCode 等主流工具。

经过多次选择后，最终配置到你的 AI 助手中。

---

### 3️⃣ 配置安装参数

- **安装范围**: Project（项目级别）
- **安装方式**: Symlink（符号链接，推荐方式）
