# Agent 4: External Views - 外部观点和批评调研

## 监管机构和政府

### 美国证券交易委员会（SEC）

#### 2023年指控
- 指控币安和CZ违反证券法
- 指控涉及13项违法行为
- 包括未经注册的证券交易、操纵市场等

#### 2025年案件结果
- 案件在2025年被撤销/和解
- CZ支付巨额罚款（约43亿美元）
- 币安承认部分违规行为
- 退出美国市场部分业务

**外部评价**:
- 支持者：监管是必要的，保护投资者
- 批评者：过度监管，扼杀创新

### 美国司法部

#### 刑事指控
- 指控CZ违反美国反洗钱法
- 指控币安未能阻止非法交易
- 包括为恐怖主义提供资金等

#### 判决结果
- CZ认罪并同意支付罚款
- 被判4个月监禁（2024年执行）
- 卸任币安CEO职务

**外部评价**:
- 支持者：法律面前人人平等
- 批评者：政治迫害，选择性执法

### 其他监管机构

#### 欧洲监管者

#### 亚洲监管者

**外部评价**:
