# 伏羲：让任何人成为你的思维顾问

> 「写不进去的那部分，才是你真正的护城河。」——但写得进去的部分，已经足够强大。

## 伏羲是什么？

伏羲是一个AI Skill，它能够将任何人的思维方式蒸馏成可运行的认知框架。无论是乔布斯、马斯克、芒格，还是福尔摩斯、钢铁侠，甚至是你的同事或你自己，伏羲都能从公开信息中提取他们的认知操作系统，生成一个可以直接对话的思维顾问。

### 核心理念

**伏羲不是复制人，是提炼思维框架。**

传统的知识传承靠的是"他做了什么"，但真正让芒格和马斯克面对同一个问题做出不同判断的，是他们**怎么看世界**。伏羲捕捉的是HOW they think，不是WHAT they said。

一个好的人物Skill是一套可运行的认知操作系统：

- **镜片**：他用什么心智模型看世界？
- **直觉规则**：他用什么决策启发式做判断？
- **表达DNA**：他怎么说话？语气、节奏、用词偏好？
- **反模式**：他绝对不会做什么？
- **输入源**：他读什么书？受谁影响？思想启蒙者是谁？
- **智识谱系**：他属于哪个思想流派？影响了谁？反对谁？
- **诚实边界**：这个Skill做不到什么？

## 工作原理

伏羲的工作流程分为五个阶段：

### Phase 0：入口分流

用户输入后，伏羲首先判断属于哪条路径：

| 用户输入 | 路径 | 示例 |
|---------|------|------|
| 明确的人名/主题 | 直接路径 | 「蒸馏芒格」「做一个费曼skill」 |
| 模糊的需求/困惑 | 诊断路径 | 「我想提升决策质量」「有没有一种思维方式能帮我看透商业本质」 |

**直接路径**：收到明确名字后，确认人物类型（真实在世/历史人物/虚拟人物）、聚焦方向、用途，然后直接进入蒸馏流程。

**诊断路径**：用户不知道该蒸馏谁时，伏羲通过1-2轮追问定位需求维度（决策/表达/创业/教学等），推荐2-3个最合适的候选方案。

### Phase 1：六路并行采集

启动6个并行Agents，每个负责不同信息维度：

| Agent | 搜索目标 | 提取重点 |
|-------|---------|---------|
| **1 著作** | 书、长文、论文、newsletter | 核心论点、自创术语、**输入源分析、智识谱系来源** |
| **2 对话** | 播客、长视频、AMA、深度采访 | 即兴回答、改变立场的瞬间、拒绝回答的问题 |
| **3 表达** | Twitter/X、微博、即刻、短文 | 高频用词、争议立场、幽默方式 |
| **4 他者** | 他人分析、书评、批评、传记 | 外部观察、批评与争议、与同行对比 |
| **5 决策** | 重大决策、转折点、争议行为 | 决策背景、事后反思、言行一致/不一致 |
| **6 时间线** | 出生/出道到现在 | 关键里程碑、思想转折点、最近动态 |

**关键创新**：Agent 1不仅提取"此人的观点是什么"，还能回答：
- ✅ 此人的观点从哪里来？（输入源）
- ✅ 此人属于哪个思想流派？（谱系定位）
- ✅ 此人影响了谁/反对谁？（思想网络）

### Phase 2：三重验证提炼

一个观点要被收录为心智模型，必须通过三重验证：

1. **跨域复现**：在≥2个不同领域/话题中出现？（不是随口一说）
2. **生成力**：能推断此人对新问题的立场？（有预测力）
3. **排他性**：不是所有聪明人都这样想？（有独特性）

三重通过 → 心智模型
仅1-2重 → 降级为决策启发式
0重 → 丢弃

### Phase 3：构建Skill

将提炼结果组装为可运行的SKILL.md：

- 3-7个心智模型（每个含名称/证据/应用/局限）
- 5-10条决策启发式（每条含场景+案例）
- 表达DNA分析
- 价值观与反模式
- 智识谱系
- 诚实边界
- 调研来源

### Phase 4：质量验证

生成后，用子Agent执行3项测试：

1. **已知测试**：选3个此人公开表态过的问题，用Skill回答，对比实际立场
2. **边缘测试**：选1个他没公开讨论过的问题，Skill应表现出适度不确定
3. **风格测试**：写一段100字分析，判断是否有此人的表达特征

## 蒸馏了什么？

伏羲从公开信息中提取七层认知结构：

| 层次 | 说明 |
|---|---|
| **怎么说话** | 表达DNA——语气、节奏、用词偏好 |
| **怎么想** | 心智模型、认知框架 |
| **怎么判断** | 决策启发式 |
| **什么不做** | 反模式、价值观底线 |
| **从哪里来** | 输入源分析——此人读什么书、受谁影响 |
| **在哪里** | 智识谱系——思想流派归属、跨领域影响 |
| **知道局限** | 诚实边界 |

## 使用场景

### 1. 决策辅助

```
用户：我同时想做自媒体、写书、做独立开发，精力完全不够

Naval：你列了三个欲望。每个欲望都是你跟不快乐签的一份合同。
不是精力不够，是合同太多。
问你自己：哪一个做起来你会忘记时间？
那个就是你的特定知识所在。
不是三选一。是先一，再一，再一。
串行复利，不是并行内耗。
```

### 2. 问题分析

```
用户：OpenAI和Anthropic谁的方向是对的？

乔布斯：你问错了。这不是一场关于方向的竞赛。
这是一场关于品味的竞赛。
OpenAI像1985年的微软。什么都做，什么都要赢。
这种策略能赢市场份额。Bill Gates就是这么赢的。
问题是——当你什么都做的时候，你不可能每件事都做到
insanely great。
```

### 3. 思维训练

- 定期与不同的Skill对话，学习多种思维框架
- 用乔布斯的"聚焦即说不"训练决策能力
- 用芒格的"多元思维模型"拓展认知边界

### 4. 内容创作

- 用Naval的视角分析商业问题
- 用费曼的方式解释复杂概念
- 用张雪峰的务实风格写职业建议
